# 📄 ATS Resume Analyzer

A web-based ATS (Applicant Tracking System) Resume Analyzer that evaluates resumes against job descriptions and provides insights like skill matching, missing keywords, grammar suggestions, and improvement tips.

---

## 🚀 Features

- 📂 Upload Resume (TXT / PDF / DOCX)
- 📝 Paste Resume manually
- 📌 Paste Job Description
- 📊 ATS Score calculation
- ✅ Matched Skills detection
- ❌ Missing Skills detection
- 💻 Technical Skills analysis
- 🤝 Soft Skills detection
- 📐 Resume formatting analysis
- 📈 Keyword density chart (Chart.js)
- ✍️ Grammar suggestions
- 🔧 Improvement recommendations
- 🌙 Dark mode toggle
- 📄 Download ATS report as PDF

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Chart.js
- PDF.js (for PDF reading)
- Mammoth.js (for DOCX reading)

## ⚙️ How It Works

1. Upload or paste your resume
2. Paste the job description
3. Click **Analyze Resume**
4. System compares:
   - Keywords
   - Skills
   - Formatting
5. Generates:
   - ATS Score
   - Suggestions
   - Charts
   - Grammar feedback
